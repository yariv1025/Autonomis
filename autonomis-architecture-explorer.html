Intents (dashed):
PLAN → Plan BUILD → Execution Loop DEBUG → Execution Loop REVIEW → Design Review
  • Router is the only entry point; it detects intent and routes to the full SDLC or a specific phase.
  • Execution Loop is deterministic: Implement → Validate → (on fail) Debug → Validate → (on pass) Review.
  • Design Review and Code Review enforce OWASP Top 10 and a performance rubric.
Autonomis — SDLC-driven orchestration for Claude Code and Cursor. Security-first (OWASP Top 10), performance-first.
